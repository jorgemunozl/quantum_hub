# Quantum Hub

Course materials and exercises for classes on **quantum computing**.

## Setup

This project targets **Python 3.13+**. With [uv](https://github.com/astral-sh/uv) or another PEP 621–aware tool:

```bash
uv sync
```

Or install in editable mode with your preferred workflow from `pyproject.toml`.

## Contents

Lecture notes, notebooks, and code will live in this repository as the course grows. Check folders at the repo root for each module or topic.
